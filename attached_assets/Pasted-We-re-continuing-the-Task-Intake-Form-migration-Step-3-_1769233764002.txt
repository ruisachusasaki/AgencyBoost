## HOW VISIBILITY CONDITIONS WORK

Each section has a `visibilityConditions` JSONB field. When null, the section always shows. When populated, the section only shows if the conditions evaluate to true.

Structure:
```json
{
  "operator": "AND",
  "conditions": [
    {
      "questionId": "uuid-of-trigger-question",
      "operator": "equals",
      "value": "Option Text Exactly As Stored"
    }
  ]
}
```

For OR logic (section shows if ANY condition is true):
```json
{
  "operator": "OR",
  "conditions": [
    { "questionId": "uuid-1", "operator": "equals", "value": "Yes" },
    { "questionId": "uuid-2", "operator": "equals", "value": "Yes" }
  ]
}
```
